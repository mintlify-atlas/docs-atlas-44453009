## Technology Stack

TechStore is built on a robust, modern technology stack:

- **Backend**: Java 21, Spring Boot 3.x, Spring Security 6, JPA/Hibernate
- **Security**: RSA + JWT authentication with stateless validation
- **Database**: MySQL/PostgreSQL with optimized queries
- **Frontend**: JavaScript SPA with Bootstrap 5 and CSS3
- **Tools**: Chart.js for analytics, Flatpickr for filters, SweetAlert2 for UX

## Architecture

The system uses a **multi-layered** decoupled architecture:

1. **REST API Layer**: Optimized endpoints for asynchronous communication
2. **Service Layer**: Transactional business logic with Spring @Service components
3. **Security Layer**: Custom filters for stateless JWT validation
4. **Data Layer**: JPA repositories with Spring Data abstractions

<Note>
TechStore follows RESTful design principles and implements CORS support for cross-origin requests.
</Note>
